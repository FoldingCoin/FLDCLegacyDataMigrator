# FLDCLegacyDataMigrator
This is a console application which converts legacy MySql database dumps into stats files for importing.

## Deployment

### App Hardware Requirements

* HDD 120GB+
* i5 Intel / Ryzen 5 AMD
* 16 GB

## Built With

* Visual Studio 2017
* ReSharper
